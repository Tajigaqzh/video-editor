# PixiJS 集成说明

## 已完成的功能

### 1. 默认轨道
- 应用启动时自动创建一个视频轨道和一个音频轨道
- 视频轨道在上方，音频轨道在下方

### 2. 拖放功能
- 从素材库拖动视频/音频文件到对应轨道
- 自动检测文件类型，只允许拖放到匹配的轨道类型
- 根据拖放位置自动计算片段在时间轴上的起始时间

### 3. 视频预览
- 使用 PixiJS 渲染引擎实现高性能视频预览
- 根据播放头位置实时显示当前帧
- 支持多轨道视频合成（按轨道顺序叠加）
- 显示片段名称和播放进度

### 4. 渲染架构
- **分层结构**：背景层 → 视频层 → 特效层 → 叠加层
- **自适应缩放**：视频自动适应预览窗口大小
- **实时更新**：播放头移动时自动更新预览内容

## 使用方法

1. **导入素材**
   - 点击素材库的"导入"按钮
   - 选择视频或音频文件

2. **添加到时间轴**
   - 拖动素材到对应的轨道（视频→视频轨道，音频→音频轨道）
   - 松开鼠标完成拖放

3. **预览视频**
   - 拖动播放头或点击时间刻度
   - 右侧预览区域会实时显示当前帧

## 技术细节

### PixiJS 配置
- 分辨率：1280x720
- 抗锯齿：开启
- 自动密度：开启
- 设备像素比：自适应

### 文件结构
- `src/components/VideoPreview/VideoPreview.tsx` - 视频预览组件
- `src/utils/pixiRenderer.ts` - PixiJS 渲染工具类
- `src/store/timelineStore.ts` - 时间轴状态管理
